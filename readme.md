### Installation

_Instructions on How to Start The App._
1. Clone the repo
   ```sh
   git clone https://github.com/Pggeeks/Live-StockScrenner-Django.git
   ```
2. install requirements.txt
    ```py
    pip freeze > requirements.txt.
    ```
3. Run the server
    ```py
    python manage.py runserver
    ```
